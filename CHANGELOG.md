# Changelog

## v0.1.97 - 2026-03-07（小）

### 追加
- 敵に回避率（`evasion`）を追加
  - `data/maps.js` の各敵に `evasion: 0.0〜1.0` で設定（省略時 = 0）
  - プレイヤー命中後に回避判定、かわすと「かわした！」ログ + EVADE 表示
  - 一括討伐モードでは敵ごとに個別回避判定
  - 敵表示に `EVA XX%` を追記（evasion > 0 の場合のみ）
- 回避率を設定した敵: 野うさぎ 10%、コウモリ 20%、スコーピオン 12%、雪うさぎ 15%、炎の精霊 15%、深海魚 18%、嵐の精霊 22%、堕天使 25%、天空の番人 15%、魔王 20%

---

## v0.1.96 - 2026-03-07（小）

### 変更
- セーブ関連ボタンをゲーム枠外の固定位置（画面右中央）にミニアイコン化
  - `SAVE_ICONS` 定数で各ボタンの画像パス・代替テキスト・ラベルを設定可能
  - `src` に画像パスを設定すると画像表示、空なら S / E / I / R のテキスト表示
  - ホバー時にツールチップでラベル（セーブ等）を表示

## v0.1.93 - 2026-03-07（小）

### 追加
- ステージ背景画像スロット（`MAP_DEFS[i].bg`）: マップごとに背景画像を設定可能（設定なしは非表示）
- マップ名横のアイコンスロット（`MAP_DEFS[i].icon`）: マップごとにアイコン画像を設定可能
- ステージ進捗グリッド: `#stage-num` を 10 マスのグリッド表示に変更
  - 現在のマスにキャラアイコン（`CHAR_ICON` 定数、未設定時は ★）を表示
  - 突破済みマスは緑背景、一括モード中は橙色で全マスハイライト
  - ボスマス（10 番目）は黄色背景
- `data/maps.js` 全マップに `bg: ""` / `icon: ""` フィールドを追加

## v0.1.9 - 2026-03-07（中）

### 追加
- **一括討伐モード**: マップのボスを 100 回倒すと、次回以降そのマップの全 10 体（ステージ 1〜9＋ボス）が一斉出現
  - プレイヤーの攻撃が全員に同時ヒット（ログに `dmg×10体` 表示）
  - 敵は全員合計 ATK で 2 秒ごとに一斉攻撃
  - 全員撃破後、ボスを経由せず直接次のマップへ
  - ステージ表示が「全 / 10」に変化
  - モンスター図鑑のマップ名に「一括×討伐数」バッジを表示（残り回数も表示）
- ボス 100 回討伐達成時にシステムログで通知

---

## v0.1.8 - 2026-03-07（大）

### 追加
- モンスター討伐数を `state.monsterKills` に保存（リセットまで保持）
- 討伐数ボーナス（100 / 500 / 999 体ごと）を各モンスターに設定（`data/maps.js` の `killBonus`）
- マップ内全モンスターを 999 体倒すとマップ制覇ボーナス（`mapBonus`）
- モンスター図鑑を画像付きカードレイアウトに刷新
  - 討伐数・ティアバッジ・次のボーナス目標を表示
  - 未到達マップ名は「???」表示、カードはグレーアウト
- 図鑑をアイテム・モンスター切り替え式の統合モーダルに変更
  - ヘッダーのボタンを「図鑑」1 つに集約、タブで切り替え

### 変更
- `data/maps.js` の全モンスター・ボスに `killBonus` / `mapBonus` を追加
- `computePlayerStats()` にモンスター討伐ボーナスとマップ制覇ボーナスを追加
- `openModal / closeModal` → `openBook / closeBook / switchBookTab` に変更

## v0.1.5 - 2026-03-07（大）

### 追加
- モンスターの出現をボス以外ランダムに変更
- アイテムドロップテーブル・ステータスをモンスターごとに個別設定可能に
  - `hpMult` / `atkMult` で各敵の強さを調整可能

### 変更
- データファイルを `game.js` から分離
  - `data/items.js` … `BONUS_THRESHOLDS` / `ITEMS` / `ITEM_MAP`
  - `data/maps.js` … `MAP_DEFS`（マップ・敵・ドロップ定義）
  - `data/shop.js` … `SHOP_DEFS`（ショップパラメーター）
  - `game.js` … ゲームロジックのみ
- `buildMaps()` を廃止、`makeEnemy(mapIndex, stageInMap)` で動的生成

# 放置ゲーム (hotige)

ブラウザで動くバニラ JS 製の放置系 RPG。

## 遊び方

`index.html` をブラウザで開くだけで動作する。サーバー不要。

キャラクターが自動で敵を攻撃し、倒すとゴールドやアイテムを獲得する。
ショップでステータスを強化しながら先のマップを目指す。

## 構成

```
hotige/
├── index.html          # メイン画面
├── style.css           # スタイル
├── game.js             # ゲームロジック
├── data/
│   ├── items.js        # アイテム・討伐ボーナス定義
│   ├── maps.js         # マップ・敵・ドロップ定義
│   └── shop.js         # ショップパラメーター
└── img/
    ├── enemy/          # 敵画像
    ├── bg/             # ステージ背景画像
    └── icon/           # マップアイコン画像
```

## ゲームシステム

### マップ・ステージ
- 10 マップ × 10 ステージ構成
- ステージ 1〜9 は通常モンスター（ランダム出現）、ステージ 10 はボス
- ボスを倒すと次のマップへ進む
- ステージ進捗はヘッダーのマス目グリッドで確認できる

### ステータス
| 項目 | 効果 |
|------|------|
| ATK  | 攻撃力（アイテムボーナスで上昇） |
| STR  | 攻撃力補正 |
| VIT  | 最大 HP（ショップで強化可） |
| DEX  | 命中率（ショップで強化可） |
| AGI  | 攻撃速度（ショップで強化可） |
| LUK  | クリティカル率（ショップで強化可） |

### 戦闘
- プレイヤーは AGI に応じた間隔で自動攻撃
- 命中率・クリティカル・敵の回避率・クリ耐性が判定に絡む
- 敵も一定間隔でプレイヤーを攻撃する
- HP が 0 になると現在のマップ先頭からやり直し（ステータス・アイテムは維持）

### アイテム・討伐ボーナス
- 敵を倒すとアイテムをドロップ（確率）
- 各モンスターを 100 / 500 / 999 体倒すと討伐ボーナス（ステータス上昇）
- マップ内全モンスターを 999 体倒すとマップ制覇ボーナス

### レアモンスター
- 各マップに 1 種類のレアモンスターが低確率（3〜4%）で出現
- 高回避・高ゴールドドロップが特徴
- マップ制覇ボーナスの計上対象外

### 一括討伐モード
- ボスを 100 回倒すとそのマップで解放
- 全 10 体が列をなして登場し、左から右へ 1 体ずつ必中・一撃必殺で撃破
- 攻撃速度は通常の 3 倍

### モンスター図鑑
- 討伐済みのモンスター情報（回避率・クリ耐性・ドロップ・討伐数・ティアバッジ）を確認できる
- 未遭遇のモンスターはマスク表示

## セーブ
- 5 秒ごとに localStorage へ自動セーブ
- 手動セーブ・ロード・リセットボタンあり（画面右端の固定アイコン）
- エクスポート / インポート機能でセーブデータを文字列として持ち運び可能

## カスタマイズ（開発者向け）

### 画像スロット
| 定数 / フィールド | 場所 | 内容 |
|---|---|---|
| `CHAR_ICON` | `game.js` | ステージグリッドのキャラアイコン |
| `PANEL_ICONS` | `game.js` | 各パネル（ステータス / アイテム / ショップ）のアイコン |
| `SAVE_ICONS` | `game.js` | セーブ系ボタンのアイコン画像 |
| `MAP_DEFS[i].bg` | `data/maps.js` | ステージ背景画像パス |
| `MAP_DEFS[i].icon` | `data/maps.js` | マップ名横のアイコン画像パス |

### 敵パラメーター（`data/maps.js`）
| フィールド | 説明 |
|---|---|
| `hpMult` / `atkMult` | HP・ATK の倍率 |
| `evasion` | 回避率（0.0〜1.0） |
| `critRes` | クリ耐性（0.0〜1.0） |
| `killBonus` | 討伐数ボーナス定義 |
| `rare` | レアモンスター定義（省略可） |
